## Instructions

Build files:

```bash
    cmake .
    make 
```

builds exe:
    - calculator
    - test

```bash
    ./<filename> <expression>
```

No parenthisis needed for expression. Will be executed from left to right.

When using the multiplication symbol * , apply delimeter \*

example

```bash
./calculator 5 \* 3 + 6
```

-> 21
